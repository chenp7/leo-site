Autoclick

A small macOS Python utility that clicks your current mouse position repeatedly. Set how many clicks you want and how fast they run.

Download Script

Run It

python3 autoclick.py -n 100 -i 0.01

After you start it, there is a 5-second delay before clicking begins.

Options

  • -n / --count: Number of clicks (default: 100)
  • -i / --interval: Seconds between clicks (default: 0.01)

Install Dependency

python3 -m pip install --user pyobjc-framework-Quartz
